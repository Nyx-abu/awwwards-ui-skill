This skill bridges the gap between functional AI code generation and high-end, premium web design. Give your agent the capability to build breathtaking 3D portfolios, deeply interactive corporate sites, and bleeding-edge landing pages with zero prompts required beyond "make it cool."

---

## 🔥 Features (Battle-Tested Version)

- **Triggers Automatically:** Reacts aggressively to prompts containing "cool ui", "amazing ui", "premium design", or "awwwards".
- **Awwwards Philosophy Built-in:** Instructs the AI on WebGL (Three.js), fluid scroll-driven animations (Lenis/GSAP), bento-box layouts, and custom magnetic cursors.
- **Deep Accessibility & Audit Layers:** Not just pretty—the AI is strictly instructed to run an **Audit Layer** checking WCAG compliance, semantic ARIA roles, Vercel guidelines, and memory leak prevention in WebGL contexts before returning the code.
- **Universal Application:** Designed to apply high-end aesthetic touches to *any* project gracefully, falling back to static UI when user preferences (like reduced motion) require it.

## 🚀 Installation

### For Antigravity
Clone this repository into your plugins directory:
```bash
git clone https://github.com/yourusername/awwwards-ui-skill.git ~/.gemini/config/plugins/awwwards-ui-skill
```

### For Claude Code / Codex
Copy the `skills/awwwards_ui_design/SKILL.md` file and incorporate it directly into your custom system prompts or skill registry.
